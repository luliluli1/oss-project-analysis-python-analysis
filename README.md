# oss-project-analysis-python-analysis
2025开源软件基础课程大作业 - 开源项目提交历史分析

## 项目介绍
本项目分析[开源项目名称]的提交历史，探索其开发模式和贡献者行为。

## 技术栈
- GitPython: 获取仓库提交历史
- pandas: 数据处理与分析
- matplotlib/seaborn: 数据可视化

## 项目结构
- src/: 源代码
- data/: 原始数据
- results/: 分析结果与图表
- docs/: 项目文档
- tests/: 测试代码

## 使用方法
1. 安装依赖: `pip install -r requirements.txt`
2. 运行主程序: `python src/main.py`
3. 查看结果: results/目录下生成的图表
